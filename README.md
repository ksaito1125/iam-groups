# IAMグループ

## Usersグループ

Usersグループは、一般ユーザのグループです。

グループに所属するユーザにユーザ自身の下記の権限を付与します。

* パスワードを変更する
* MFAデバイスを管理する
* アクセスキー、SSHキーを管理する
